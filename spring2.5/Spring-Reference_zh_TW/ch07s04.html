7.4. Spring裡的Advisor API

在Spring裡，一個Advisor是一個僅僅包含一個通知物件和與之關聯的切入點表達式的切面。

除了引入這種特殊形式，任何通知器（advisor）都可以和任何通知一起工作。 org.springframework.aop.support.DefaultPointcutAdvisor是最常用的通知器類別。例如，它可以和 MethodInterceptor，BeforeAdvice或者 ThrowsAdvice一起使用。

在Spring裡有可能在同一個AOP代理裡混合通知器和通知型別。 例如，你可以在一個代理配置裡使用一個攔截環繞通知，一個異常通知和一個前置通知：Spring將負責自動創建所需的攔截器鏈。

6.7. 以程式方式創建@AspectJ代理

除了在配置檔案中使用<aop:config>或者<aop:aspectj-autoproxy>來宣告切面。 同樣可以通過程式方式來創建代理來通知目標物件。關於Spring AOP API的詳細介紹， 請參看下一章。這裡我們重點介紹自動創建代理。

類別org.springframework.aop.aspectj.annotation.AspectJProxyFactory可以為一個或多個 @AspectJ切面通知的目標物件創建一個代理。該類別的基本用法非常簡單，範例如下。請參閱Javadoc獲取更詳細的資訊。 

// create a factory that can generate a proxy for the given target object
            AspectJProxyFactory factory = new AspectJProxyFactory(targetObject); 
            
            // add an aspect, the class must be an @AspectJ aspect
            // you can call this as many times as you need with different aspects
            factory.addAspect(SecurityManager.class);
            
            // you can also add existing aspect instances, the type of the object supplied must be an @AspectJ aspect
            factory.addAspect(usageTracker);	
            
            // now get the proxy object...
        MyInterfaceType proxy = factory.getProxy();

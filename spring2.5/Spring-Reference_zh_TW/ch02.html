2.1. 簡介

如果你已經用了一段時間的Spring Framework，那你將發現Spring經歷了兩次大的修訂: 一次是2006年10月發佈的Spring 2.0, 另一次是2007年11月發佈 Spring 2.5 。

Java SE 與 Java EE 支持

Spring Framework繼續保持與所有Java版本的相容性 - 從Java 1.4.2開始(包括1.4.2)。這意味著spring對Java1.4.2,Java 5和 Java 6都支持， 但是Spring Framework的一些高層級功能無法在1.4.2中使用。從Spring 2.5起,Spring框架完全支持Java 6，而Spring 2.0則對Java 5支持比較好。

此外，Spring延續了對J2EE 1.3及更高版本的相容性，同時對Java EE 5提供完全支持。也就是說，Spring可以繼續在應用伺服器中運行，包括 BEA WebLogic 8.1, 9.0, 9.2 和 10, IBM WebSphere 5.1, 6.0 和 6.1, Oracle OC4J 10.1.3 和 11, JBoss 3.2, 4.0 和 4.2, 以及 Tomcat 4.1, 5.0, 5.5 和 6.0, Jetty 4.2, 5.1 和 6.1, Resin 2.1, 3.0 和 3.1 還有 GlassFish V1 和 V2.

本章是對Spring 2.0與2.5新特性與改進特性的嚮導。我們希望提供一個高階的概觀使那些有經驗的Spring架構師與開發人員能很快熟悉Spring 2.x的新功能。 如果想瞭解關於特性更多更深層的資訊，請參考在本章裡超連接的相應部分。
